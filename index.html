a = b * 2

  • a b: Son variables, sirven para almacenar datos de manera temporal
  • = *: Son opereadores, sirven para realizar acciones sobre variables o datos estáticos, en este caso asignación el = y operación aritmética el *
  • 2: Es un valor literal, no estan almacenado en variable

Operadores

						
							var a = 20;
							a = a + 1;
							a = a * 2;
							console.log( a ); // 42
  • En javascript primero se calcula el valor de los operandos y luego se asigna.
  • Se tienen que declaran primero las variables si no no se pueden operar con ellas.

Tipos de operadores

  • Asignación
  • Matemáticos
  • Asignación compuesta
  • Incrementar/Decrecer
  • Acceder a propiedades del objeto
  • Igualdad
  • Comparativos
  • Lógicos

Operadores de asignación

Es el carácter =

						
									var pitufo = 'Esto es un pitufo';
									alert(pitufo);
									pitufo = 3; // Ahora pitufo va a ser un número
									alert(pitufo);
									pitufo = {
										altura: 0.1,
										color: 'azul'
									}; // Ahora pitufo va a ser un objeto
									alert('Soy un pitufo y mido ' + pitufo.altura + ' metros y soy 
									de color ' + pitufo.color);

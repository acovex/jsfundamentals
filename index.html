Operadores matemáticos

  • + Adición: Ej: alert(2+3)
  • - Substracción: Ej. alert(52-15)
  • * Multiplicación: Ej. alert(5*6)
  • / División: Ej. alert(12/5)

Operadores de asignación compuesta

Son operadores compuestos, primero realizan la operación matemática y luego la asignación.

  • += Adición: Ej: var a = 2; a+=3; alert(a)
  • -= Substracción: Ej. var a = 52; a-=15; alert(a)
  • *= Multiplicación: Ej. var a = 5; a*=6; alert(a)
  • /= División: Ej. var a = 12; a/=5; alert(a)

Operadores de incrementar decrecer

Para facilitar la adición o decrecimiento de valor 1. Operación más asignación.

  • ++ Incrementar: Ej: var a = 2; a++; alert(a)
  • -- Decrecer: Ej. a = 2; a--; alert(a)

Operadores de igualdad

  • == Igual sin restricción de tipo
    • alert("2 == '2': " + (2 == '2'));
    • alert("2 == 2: " + (2 == 2));
    • alert('false == 0 : ' + (false == 0));
  • === Igual con restricción de tipo
    • alert("2 === '2': " + (2 === '2'));
    • alert("2 === 2: " + (2 === 2));
    • alert('false === 0 : ' + (false === 0));

Operadores de igualdad

  • != Distinto a sin restricción de tipo
    • alert("2 != '2': " + (2 != '2'));
    • alert("2 != 2: " + (2 != 2));
    • alert('false == 0 : ' + (false == 0));
  • !== Distinto a con restricción de tipo
    • alert("2 !== '2': " + (2 !== '2'));
    • alert("2 !== 2: " + (2 !== 2));
    • alert('false === 0 : ' + (false === 0));

Operadores de comparación

  • < Menor a, <= Menor o igual a
    • alert(' 2 < 2 : ' + (2 < 2));
    • alert(' 2 <= 2 : ' + (2 <= 2));
  • > Mayor a, >= Mayor o igual a
    • alert(' 2 < 2 : ' + (2 < 2));
    • alert(' 2 <= 2 : ' + (2 <= 2));

Operadores lógicos

  • && (and)
    • var a = 3; var b = 4; alert(a && b ? '¡Existen!' : 'Alguno no existe');
    • var a = 3; var b = null; alert(a && b ? '¡Existen!' : 'Alguno no existe');
  • || (or)
    • var a = 3; var b = null; alert(a || b);
    • var a = null; var b = 5; alert(a || b);

Bloques de código

Cuando necesitamos una serie de operaciones juntas las agrupamos en lo que llamamos bloque. Los bloques estan definidos dentro de un par de llaves { .. } 

					
						var montante = 99.99;
						// un bloque génerico
						{
							montante = montante * 2;
							console.log( montante ); // 199.98
						}

Condicionales: switch

Sirven para acceder a un bloque de código u otro en caso de que se cumpla un determinado valor o no para el caso correspondiente.

Condicionales: switch

						
						var inputsForm = document.getElementById('colorPitufoForm').elements;
						var colorPitufoReal = inputsForm['colorRealPitufo'].value;
						var colorPitufoElegido = inputsForm['colorPitufo'].value;
						switch(colorPitufoElegido) {
							case 'azul': 
								if(colorPitufoReal === 'azul') {
									alert('¡Correcto! Los pitufos siempre han sido azules');
								} else {
									alert('¡Error! Este es un raro caso de un pitufo malvado');
								}

							case 'rojo': 
								if(colorPitufoReal === 'rojo') {
